# Smart Inventory Management System

A full-stack inventory management application that helps businesses manage products, track stock levels, and perform CRUD operations efficiently.

## Features
- **Product Management:** Add, update, delete, and view products.
- **Stock Monitoring:** Track stock levels and get low-stock alerts.
- **Pagination & Search:** Easily find products with search and paginated views.
- **User Roles:** Admin-specific features (like managing users and settings).
- **Responsive UI:** Built with React and Tailwind CSS for modern and responsive interfaces.
- **REST API Backend:** Powered by Spring Boot and connected to MySQL database.
- **Dark Mode Support:** Switch between light and dark themes.

## Tech Stack
- **Frontend:** React, Tailwind CSS, React Router, Axios
- **Backend:** Spring Boot, Spring Data JPA, MySQL, Lombok
- **Build Tools:** Maven (Backend), Node.js/NPM (Frontend)
